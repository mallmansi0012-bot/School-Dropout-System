# School Dropout Monitoring System (C Project)

## 📌 Description
This project is a simple School Dropout Monitoring System written in C.  
It helps track student attendance and identify students at risk of dropping out.

## 🚀 Features
- Add student details
- Track attendance
- Identify dropout risk
- Search student by ID
- Display all students

## 🧠 Logic
Students with attendance below 75% are marked as "At Risk".

## 🛠️ Technologies Used
- C Programming
- Structures
- Arrays
